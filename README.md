# 🖥️ Linux 服务器监控系统

一个简洁高效的 Linux 服务器实时监控系统，支持通过 SSH 远程监控多台服务器。

## ✨ 功能特性

- 📊 **实时 Web 仪表盘** - 漂亮的可视化界面
- 🌙 **亮色/暗色主题** - 一键切换，保护眼睛
- 🔄 **自动刷新** - 数据每 10 秒自动更新
- ➕ **服务器管理** - 支持添加/删除服务器
- 📈 **多项监控指标**
  - 内存使用情况
  - 磁盘空间
  - CPU 负载
  - 网络流量
  - 系统运行时间
- 🔐 **SSH 安全连接** - 无需在服务器安装任何代理

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install paramiko pyyaml pybars3 requests python-dateutil flask
```

### 2. 配置服务器

编辑 `servers.yml`：

```yaml
servers:
  "我的服务器":
    driver: ssh
    config:
      host: 192.168.1.100
      username: root
      password: your_password
      port: 22
    monitors:
      - type: memory
      - type: disk
      - type: loadavg
      - type: network
        config:
          match: "ens*"
```

### 3. 启动监控

```bash
./start-monitor.sh
```

然后打开浏览器访问：**http://localhost:8080**

## 📖 使用说明

### 添加服务器

1. 点击右上角 **「➕ 添加服务器」** 按钮
2. 填写服务器信息：
   - 服务器名称
   - 主机地址（IP 或域名）
   - SSH 端口（默认 22）
   - SSH 用户名
   - SSH 密码
3. 点击 **「✅ 添加服务器」**

### 删除服务器

1. 点击 **「➕ 添加服务器」** 按钮
2. 在服务器列表中找到要删除的服务器
3. 点击 **「🗑️ 删除」** 按钮

### 切换主题

点击右上角的 **「☀️」** 或 **「🌙」** 按钮即可切换亮色/暗色主题。

## 🛠️ 命令行工具

```bash
# 查看监控摘要（Markdown格式）
python3 ./sshmon summary servers.yml -f md

# 生成HTML报告
python3 ./sshmon summary servers.yml -f modern > report.html

# 检查告警（用于定时任务）
python3 ./sshmon check servers.yml

# 启动Web服务
python3 ./sshmon serve servers.yml --port 8080 --refresh 10
```

## 🔧 支持的监控项

| 监控项 | 说明 | 配置示例 |
|--------|------|----------|
| memory | 内存使用情况 | `- type: memory` |
| disk | 磁盘空间 | `- type: disk` |
| loadavg | CPU负载 | `- type: loadavg` |
| network | 网络流量 | `- type: network` |
| system | 系统信息 | `- type: system` |
| tcp | 端口检查 | `- type: tcp` |

## ⚠️ 注意事项

1. **SSH 连接**：确保 SSH 用户名和密码正确
2. **网络**：服务器和监控机器之间需要网络互通
3. **权限**：SSH 用户需要有读取 `/proc` 文件系统的权限
